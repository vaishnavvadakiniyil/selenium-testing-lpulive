LPU UMS Login Automation using Selenium WebDriver

This project automates the login functionality of the LPU UMS (University Management System) using Java Selenium WebDriver.
It opens the UMS login page, enters the username and password, waits for manual CAPTCHA input, and performs the login action.

Opens LPU UMS login page automatically

Enters registration number and password

Handles dynamic elements with Implicit Wait

Provides 30 seconds for manual CAPTCHA input

Clicks the login button to complete authentication
